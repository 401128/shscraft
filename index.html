Nerd Info:

The only reason why the Minecraft URL doesn't work on school Wi-Fi is because it starts with Http instead of Https. The network is configured to only allow secure connections using "https" which encrypts data, preventing you from accessing non-secure websites that only use "http" - this is a common security measure implemented by schools to protect student data and more.

This issue will be resolved in the future as Evan (Owner of Minecraft server) is trying to figure out how to get it vertified.
